# cpp_console_lib - A small console library in C++
This is a small console library I made in C++ because I didn't find any ones that I liked
## Functions
- Colors
  - 16 bit colors
  - True color RGB
  - Text types
    - Default
    - Light
    - Italic
    - Bold
- Eraser
  - Erase lines
  - Erase screen
- Move
  - Move cursor (exact)
  - Move cursor (relative)
- Cursor visibility
- Window size stuff
  - Right bound
  - Center bound
  - Header
